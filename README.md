# This is wonho's TodoList

## Installing
Using gradle
```bash
$ git clone https://github.com/takeelf/todolist.git
$ gradle build
$ java -jar build/libs/todolist-0.0.1-SNAPSHOT.jar
```
Connect localhost:8080

## Stacks

- Spring boot
- Gradle
- Vue.js
- H2 memory DB

## Features
- Add Todo
- List Todo
- Update Todo
- Add reference Todo
- Delete Todo
- Pagination
